# MiniSTL
本仓库原是《STL源码剖析》一书的代码合集，并在其基础上运用部分C++17(主要是C++11) 特性进行了代码改写。

> **说明：**代码基本上来自[zsmj2017/MiniSTL](https://github.com/zsmj2017/MiniSTL)

### 完成进度

- [x] Allocator (空间适配器)
- [x] Iterator (迭代器与traits编程技法)
- [x] Sequence (序列式容器)
- [ ] Associative (关联式容器)
- [x] Algorithm (算法)
- [x] Functor (仿函数)
